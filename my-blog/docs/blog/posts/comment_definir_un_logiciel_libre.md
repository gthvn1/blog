# Comment définir un logiciel libre...

[Richard Stallman](https://stallman.org/), le fondateur du mouvement du logiciel libre, définit un logiciel libre comme étant un logiciel qui respecte la liberté des utilisateurs.
Selon lui, un logiciel est libre si ses utilisateurs ont les quatre libertés essentielles suivantes :

1. **Liberté d'exécuter le programme comme vous le souhaitez, pour n'importe quel usage (liberté 0)**.
   - Cela signifie que vous pouvez utiliser le programme à n'importe quelle fin, sans restriction.

2. **Liberté d'étudier le fonctionnement du programme et de le modifier pour qu'il fasse ce que vous souhaitez (liberté 1)**.
   - Pour cette liberté, l'accès au code source est une condition préalable.

3. **Liberté de redistribuer des copies du programme, donc d'aider votre voisin (liberté 2)**.
   - Vous pouvez partager des copies du logiciel avec d'autres personnes.

4. **Liberté de distribuer des copies de vos versions modifiées à d'autres (liberté 3)**.
   - En faisant cela, vous pouvez donner à toute la communauté la possibilité de bénéficier de vos modifications. L'accès au code source est également une condition préalable pour cette liberté.

Ces libertés ont pour but de garantir que les utilisateurs puissent contrôler les logiciels qu'ils utilisent, plutôt que d'être contrôlés par ceux-ci. C'est un aspect fondamental de l'éthique du logiciel libre promue par Stallman et la Free Software Foundation ([FSF](https://www.fsf.org/)).
